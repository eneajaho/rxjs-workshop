# RxJS Workshop

### Clone the project
```bash
git clone https://github.com/eneajaho/rxjs-workshop.git
```

### Install dependencies
```bash
npm install
```

### Run the app
Start the app
```bash
ng s -o
```
Start fake backend
```bash
npm run start-db
```
